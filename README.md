# NunasDownload - 下载增强

本插件是在CoreDownload插件的基础上逆向开发出来的。为 WordPress 提供多功能下载面板，支持多网盘下载地址管理、在CoreDownload的基础上增加了微信公众号验证码下载功能，优化了整个插件的UI界面。

## 功能特性

### 📥 下载地址管理
- 在文章编辑页添加下载 Meta Box，支持 17 种网盘/存储服务
- 网盘列表：本地下载、百度网盘、阿里云盘、夸克网盘、蓝奏云、123盘、微云、迅雷、GitHub、诚通网盘、UC、文叔叔、坚果云、天翼云、和彩云、OneDrive、360网盘
- 每个下载项支持：网盘选择（下拉框）、下载链接、提取码
- 可添加多个下载地址，布局紧凑

### 🔐 微信公众号验证码下载
- 设置验证码后，用户需输入正确验证码才能获取下载链接
- 支持展示公众号二维码引导关注
- 验证通过后 Cookie 记录状态（2 小时有效），无需重复验证
- 关键词提示功能

### 🎨 前端展示
- 响应式网格布局（4 列 → 3 列 → 2 列 → 1 列）
- 网盘 SVG 图标显示
- 提取码一键复制
- 主题颜色可自定义
- 投诉链接、版权声明

### ⚙️ 后台设置
- 全局开关
- 主题颜色 / 边框圆角
- 公众号二维码上传
- 投诉链接 / 版权声明

## 安装

1. 下载插件 ZIP 包
2. 在 WordPress 后台 → 插件 → 安装插件 → 上传插件，选择 ZIP 文件上传
3. 启用插件
4. 进入后台「下载增强」设置页面进行全局配置
5. 在文章编辑页中使用下载 Meta Box 添加下载地址

## 最低要求

- WordPress 5.0+
- PHP 7.4+

## 更新日志

### v1.1.0 — 2026-07-04

#### 🐛 Bug 修复
- **修复命名空间兼容性问题：** 所有 WordPress 全局函数调用添加 `\` 前缀，解决某些 PHP 环境下命名空间函数回退失败导致的白屏/致命错误
- **修复插件启动时调用 `wp_create_nonce()`：** 将 nonce 生成推迟到 `admin_enqueue_scripts` 钩子，避免插件在 WordPress 未完全加载时调用未定义函数
- **修复 AJAX 方法名大小写问题：** `sanitize_key()` 将方法名转为小写导致白名单匹配失败，报"方法不存在"，改为 `sanitize_text_field()`
- **修复 `{url}` 占位符不被替换：** DOMDocument::saveHTML() 将 `{` `}` 编码为 HTML 实体，导致模板占位符替换失败，链接跳转到 `/{url}`，切换为直接字符串替换模式
- **修复链接缺少协议前缀：** 管理员输入不带 `https://` 的链接时自动补全，防止浏览器作为相对路径处理
- **补充 hecai.svg（和彩云）图标**
- **删除 `editor.js` 死代码**

#### 💄 优化
- **文章编辑页下载列表 UI 重做：** 网盘选择从标签页改为自定义下拉选择器，向上展开，显示 6 项可滚动
- **移除资源描述字段：** 布局更紧凑，单行显示下载链接 + 提取码
- **CSS 默认值与 PHP 默认值同步**
- **验证弹窗关键词添加双引号：** `回复关键词 "XXXX" 获取验证码`

## 项目结构

```
nunas-download/
├── index.php                    # 插件入口

│   ├── Config.php               # 全局配置

│   └── Tools.php                # 工具函数
├── pages/
│   └── admin-set.php            # 后台设置页入口
├── template/components/
│   ├── download-warp.php        # 前端下载面板外壳
│   └── download-url-item.php    # 单个下载项模板

    ├── css/                     # 样式文件
    ├── js/                      # JavaScript 文件
    └── img/
        └── icon/                # 各网盘 SVG 图标
```
